## [1.0.0] - 2024-01-13

### Added
- 초기 프로젝트 구조 설정
- 회원가입 기능
  - 이름, 학년 반 번호, 비밀번호로 가입
  - 중복 아이디 검사
  - 입력값 검증
- 로그인 기능
  - 아이디(이름)와 비밀번호로 로그인
  - 구글 계정 OAuth 로그인 (Firebase)
  - 세션 기반 인증
- 로그아웃 기능
- 게시판 기능
  - 공지사항 게시판 (읽기 전용)
  - 질의응답 게시판 (읽기 전용)
- 사용자 친화적 UI/UX
  - 밝고 예쁜 컬러 팔레트
  - 반응형 디자인
  - 부드러운 애니메이션
  - 재미있는 이모지
- 보안
  - 비밀번호 최소 길이 검증
  - Firebase 인증
  - 환경 변수 관리

### Changed
- 프로젝트 구조 정리
  - templates 폴더로 HTML 파일 분류
  - static 폴더 생성 (향후 CSS/JS 분리용)

### Security
- SECRET_KEY를 환경 변수로 관리
- 입력값 검증 추가
- CSRF 보호 (Flask-WTF 예정)
